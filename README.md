# Offscript Labs - Modern Homepage

A sleek, modern homepage for Offscript Labs featuring a dark theme and elegant typography that matches your brand identity.

## Features

- **Dark Theme**: Sophisticated black and dark gray color scheme
- **Responsive Design**: Fully responsive across all devices
- **Modern Typography**: Uses Playfair Display (italic) for "Offscript" and Inter for "Labs" to match your logo
- **Smooth Animations**: Fade-in effects, hover animations, and smooth scrolling
- **Interactive Elements**: Button ripple effects and parallax scrolling
- **Professional Layout**: Clean, minimalist design with proper spacing

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality and animations
- `README.md` - This documentation

## How to Use

1. **Local Development**: Simply open `index.html` in your web browser
2. **Web Hosting**: Upload all files to your web server
3. **Customization**: Edit the content in `index.html` and styling in `styles.css`

## Design Elements

### Typography
- **"Offscript"**: Playfair Display italic - elegant serif font
- **"Labs"**: Inter sans-serif - modern, clean font
- **Body Text**: Inter for readability

### Color Scheme
- **Background**: Dark gradient from #0a0a0a to #1a1a1a
- **Text**: White (#ffffff) and light gray (#cccccc)
- **Accents**: Subtle white borders and hover effects

### Interactive Features
- Smooth scrolling navigation
- Fade-in animations for sections
- Hover effects on buttons and cards
- Typing animation for hero subtitle
- Parallax scrolling effect
- Button ripple effects

## Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
body {
    background: linear-gradient(135deg, #0a0a0a 0%, #1a1a1a 100%);
    color: #ffffff;
}
```

### Updating Content
Modify the text content in `index.html`:
- Hero section text
- About section
- Services offered
- Contact information

### Adding Sections
Follow the existing pattern in `index.html`:
```html
<section id="your-section" class="section">
    <div class="section-content">
        <h2 class="section-title">Your Title</h2>
        <p class="section-text">Your content here</p>
    </div>
</section>
```
